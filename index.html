Conway’s Game of Life

Generation: 0
Alive Cells: 0
Speed: 5
Size: 30x30
Color: OFF

The Game of Life is a 2D Cellular Automata 0 Player Game!

  • Click on any cells to make it come alive, or press the add 'Random' button
  • Next click the start button to start the game of life which uses the following rules:
    1. Any live cell with fewer than two live neighbors dies, as if by underpopulation.
    2. Any live cell with two or three live neighbors lives on to the next generation.
    3. Any live cell with more than three live neighbors dies, as if by overpopulation.
    4. Any dead cell with three live neighbous becomes a live cell, as if by reproduction.
    5. Edges are dead zones and are NOT counted as neighbors (dead or alive)
  • The step button will move you forward one generation at a time
  • Use the Slower and Faster buttons to change the game speed
  • Use the Smaller and Bigger buttons to change the number of cells on the grid
